Les commentaires

Les commentaires dans la programmation permettent de se referencier dans le code et de comprendre de quoi parle le code apres avoir laisser pendant un bout de temps.
Il y'a deux types de commentaires en JavaScript:

les commentaires mono-lignes

ceux sont des commentaires qui se font sur une ligne (// je suis un commentaire mono-ligne)

les commentaires multi-lignes

sont des commentairesqui se font sur plusieurs lignes (/* je suis * µn commentaire * multi-ligne */ ).

les variables

Une variable JavaScript est un conteneur qui permet à stocker temporairement une information comme un nombre ou une chaine de caracteres.
comme son nom l'indique le propre d'une variable est de pouvoir la variée c'est à dire de pouvoir stockée diferentes valeur dans le temps en écrasant à chaque fois sa valeur precédente.
